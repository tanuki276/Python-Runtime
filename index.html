Python C拡張 花火アニメーション 手順まとめ

必要なモジュールのインストール

pkg update && pkg install clang python make unzip git

リポジトリクローン

git clone https://github.com/tanuki276/Python-Runtime.git
cd Python-Runtime/termux-python-ext-sample

ビルド

python setup.py build

インストール

python setup.py install --user

花火アニメーション実行

python3 -c "import myext; myext.fireworks(10)"

ポイント

  • C拡張モジュール myext で高速な花火アニメーションを実現
  • ANSIエスケープシーケンスによるカラー＆カーソル制御で滑らかな表示
  • Pythonから myext.fireworks(repeat) で花火を好きな回数打ち上げ可能
  • 画面クリア・カーソル非表示でちらつき軽減
  • ターミナルでカラフルな花火を楽しもう🔥
